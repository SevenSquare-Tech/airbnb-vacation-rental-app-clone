Airbnb Vacation Rental App Clone

Project Overview:
This project is a clone of an Airbnb-style vacation rental web application built using React. It allows users to browse listings, select dates, and explore rental options with a modern UI.

Tech Stack:

- React (v16.13.1)
- React Router DOM
- React Date Range
- Date-fns
- React Scripts (Create React App)

Installation Steps:

1. Clone the repository:
   git clone https://github.com/SevenSquare-Tech/airbnb-vacation-rental-app-clone.git

2. Navigate into the project directory:
   cd airbnb-vacation-rental-app-clone

3. Install dependencies:
   npm install

4. Start the development server:
   npm start

The app will run on:
http://localhost:3000

Available Scripts:

- npm start
  Runs the app in development mode.

- npm run build
  Builds the app for production.

- npm test
  Launches the test runner.

- npm run eject
  Ejects the app configuration (irreversible).

Project Structure:

- /src
  Contains all React components and logic

- /public
  Static assets and HTML template

Features:

- Home page with listings
- Date range picker for booking
- Navigation using React Router
- Responsive UI design

Browser Support:

Production:

- > 0.2%
- Not dead
- Not Opera Mini

Development:

- Latest Chrome, Firefox, Safari

Notes:

- This project uses an older version of React (16.x). Consider upgrading if needed.
- Make sure Node.js and npm are installed before running the project.
